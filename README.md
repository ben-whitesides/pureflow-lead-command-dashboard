# PureFlow Lead Command Dashboard

Static GitHub Pages dashboard for PureFlow commercial water cooler prospecting routes, including Davis County, Sandy-to-Nephi, Salt Lake suburb add-on routes, a public-footprint stability / credit-proxy tab, and local CRM Mode for notes, statuses, follow-ups, and CSV exports.

The dashboard is intentionally self-contained: open `index.html` or serve the repo through GitHub Pages.
